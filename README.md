# 📖 Project Description

GoZon is a full-stack e-commerce web application that provides a fast, responsive, and user-friendly online shopping experience. Users can browse products, search by category, view product details, add items to their cart, manage wishlists, and securely place orders. The application features authentication, dynamic product management, and an intuitive interface designed for both desktop and mobile devices.

# 📂 Installation & Setup

## 1. Clone the repository

```bash
git clone https://github.com/your-username/gozon-client.git
```

## 2. Navigate to the project directory

```bash
cd gozon-client
```

## 3. Install dependencies

```bash
npm install
```

## 4. Create a `.env.local` file

Add your environment variables.

Example:

```env
VITE_API_URL=your_server_url

VITE_FIREBASE_API_KEY=your_key
VITE_FIREBASE_AUTH_DOMAIN=your_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender
VITE_FIREBASE_APP_ID=your_app_id
```

---

## 5. Start the development server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```
